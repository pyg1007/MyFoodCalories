## Kotilin Android Client

### 사용 기술
>- DI(Hilt)
>- Retrofit2
>- Mvvm(with repository, usecase)

## Python Flask Server(with DL)

Flask 기반 Rest-api 서버로 객체 탐지 및 분류 하여 결과를 리턴해줍니다.

### 사용기술
>- Flask
>- Pytorch
>   + Yolov3 (객체 탐지 및 분류)
>   + ResNet (음식양추정)
>- OpenCV
